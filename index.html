Welcome! First-Time Setup

Enter your JSONBin credentials to create your cloud database. This is required only once per device.

🔐 How to get JSONBin credentials (FREE):

1. Go to jsonbin.io and sign up
2. Go to Dashboard → API Keys
3. Click "Create Access Key" - Copy the key (starts with $2b$)
4. Go to Dashboard → Create Bin
5. Name it "FabricFlow" and click Create
6. Copy the Bin ID from the URL
